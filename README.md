## Принятие решений по ценовой политике компании с помощью разных методов ценообразования, определение скидочных программ.

### 2. Рекомендации по ценообразованию
Метод 1: Конкурентное ценообразование
* Установить цену 65–68 тыс. ₽ (ближе к рекомендованной 66 тыс. ₽)
* Позиционирование: цена ниже ГМПФ (77 тыс. ₽) при сравнимом качестве

Метод 2: Затратное + целевая рентабельность
